==============

###Its a jekyll app

Rename hook.sample.json to hook.json and put your git web hook secret key there.

Install all dependencies

```sh
npm install
```

Install gulp

```sh
npm install -g gulp
```

Install phantom.js

```sh
brew install phantomjs
```

Install jekyll

```sh
gem install jekyll
```

Generate full site:

```sh
gulp
```

Run tests

```sh
npm test
```
